# Guardian Portraits

Place guardian portrait images here.

**Recommended size:** 200x200px or 400x400px
**Format:** .jpg or .png

**Example files:**
- `stella.jpg`
- `vawn.png`
- `tiberius.jpg`
- `maestra.png`

Then update `data/guardians.json`:
```json
"portrait": {
  "type": "image",
  "value": "guardians/stella.jpg",
  "show_name": false
}
```
