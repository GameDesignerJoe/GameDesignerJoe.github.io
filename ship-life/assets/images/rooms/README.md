# Room Backgrounds

Place room background images here.

**Recommended size:** 1920x1080px (Full HD)
**Format:** .jpg or .png

**Example files:**
- `bridge.jpg` - Mission Computer room
- `engineering.jpg` - Workstation Room
- `portal.jpg` - Planetfall Portal
- `quarters.jpg` - Quarters/Stats room

Then update `data/rooms.json`:
```json
"background": {
  "type": "image",
  "value": "rooms/bridge.jpg"
}
```
