# Workstation Visuals

Place workstation icon/visual images here.

**Recommended size:** 200x200px or 256x256px
**Format:** .png or .jpg

**Example files:**
- `fabricator.png`
- `stellar_forge.png`
- `nano_lab.png`

Then update `data/workstations.json`:
```json
"visual": {
  "type": "image",
  "value": "workstations/fabricator.png",
  "show_name": false
}
```
