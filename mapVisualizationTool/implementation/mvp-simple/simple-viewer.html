Map Visualization Tool

A powerful tool for creating and visualizing map content distributions

Overview

The Map Visualization Tool is designed to help game designers, world builders, and content creators generate and visualize map distributions. It provides an intuitive interface for configuring map parameters, defining content types, and analyzing the resulting distributions.

Key Features

  • Three-Panel Interface: Input Panel, Map Visualization Panel, and Analysis Panel
  • Map Configuration: Set width, height, and cell size for the map
  • Content Type Management: Create, edit, and remove content types with custom colors and distribution percentages
  • Grid Visualization: View the generated map with a grid
  • Content Distribution Algorithm: Generate maps based on specified content type percentages
  • Analysis Metrics: View distribution statistics and warnings
  • Export Functionality: Export the generated map as a PNG image

Default Configuration

Parameter Value Description
Width 60 cells Number of cells horizontally
Height 50 cells Number of cells vertically
Cell Size 10 pixels Size of each cell in pixels
Grid Padding 0 pixels No padding between cells for a continuous appearance

Running the Full Application

To access all features of the Map Visualization Tool, you need to run the application using the Vite development server:

  1. Navigate to the project directory
  2. Install dependencies: npm install
  3. Start the development server: npm run dev
  4. Open http://localhost:5173 in your browser

This will provide you with the complete interactive experience, including real-time map generation and analysis.

Technologies Used

  • React: A JavaScript library for building user interfaces
  • TypeScript: A typed superset of JavaScript
  • Vite: A fast build tool and development server
  • HTML5 Canvas API: For drawing the grid and map visualization
