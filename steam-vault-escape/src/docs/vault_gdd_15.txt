Generated by clicking unlocked game cards
Each game gives points = its Steam playtime hours

Progression Balance
Target Loop Timing:

Start with 8 unlocked games (player's most-played)
Click for ~5-10 minutes â†’ unlock 2-3 new games
~15-20 min total â†’ all games MAXED OUT
Forced choice: Keep clicking for nothing OR play new game
Play new game (30+ min commitment)
Return â†’ Everything reset + big key reward
Can now unlock 3-5 better games

Goal: ~30 min of clicking before forced to play something new.
Key Scarcity:
Playing a 90-rated game for 30+ min = 90 Keys
That's enough to unlock:

1 epic game (90+ rating)
OR 2 good games (75-85 rating)
OR 3 decent games (60-75 rating)

Each unlock should feel meaningful, not disposable.

typescriptfunction GameCard({ game, progress }) {
  const isLocked = !unlockedGames.includes(game.appid);
  const isMaxed = progress?.isMaxed || false;
  const isClickable = !isLocked && !isMaxed;
  
  if (isLocked) {
    return <LockedGameSilhouette game={game} />;
  }
  
  if (isMaxed) {
    return <MaxedOutCard game={game} />;
  }
  
  return <ClickableGameCard game={game} progress={progress} />;
}
