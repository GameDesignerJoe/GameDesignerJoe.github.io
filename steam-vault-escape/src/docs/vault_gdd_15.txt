Steam Vault Escape - Game Design Document v1.5
Core Philosophy Change
v1.0 Problem: Players could unlock 50+ games instantly by clicking. No incentive to leave the site. Overwhelming choice paralysis with 300+ visible games.
v1.5 Solution: Gacha-style game pool with limited visibility (5 games at a time), forced drain mechanics, and dual-currency scarcity that pushes players to actually play Steam games.

The Three-Pool System
Pool 1: Game Library (Unlocked & Clickable)
Definition: Games the player has unlocked in the idle game and can actively click for Collection Power.
Behavior:

Visible in "Game Library" section
Clickable to generate Collection Power
Each click generates power = Steam playtime hours

If none match: Any game with 5-10 hours played
If still none: Lowest playtime game in library


Pool 2: Hidden Pool (Played but Locked)
Definition: All games the player has played on Steam but not yet unlocked in the idle game.
Behavior:

Completely invisible to player (hidden pool)
Feeds the 5-slot shop via random draws
Games revealed only when drawn into shop slots
Requires Collection Power to unlock â†’ moves to Pool 1
New games enter this pool when Key Games are played

Starting State: All played Steam games minus:

The 1 starting unlocked game
The 5 games initially drawn into shop


Pool 3: Key Games (Never Played)
Definition: All Steam games with 0 minutes playtime.
Behavior:

Displayed in "KEY GAMES" section as visual gallery

Game now eligible for future shop draws



UI Display:

No unlock button (can only unlock by playing on actual Steam)

Draw New Game Mechanic
The Flow:
Step 1: Player clicks empty shop slot

Cost: 10 Liberation Keys
Opens "Draw New Game" UI overlay

Step 2: Slot machine interface

Shows 3 card backs (ğŸ´ ğŸ´ ğŸ´)
Player clicks one card back
Behind the scenes: Game is predetermined via weighted random from Pool 2

Step 3: Reveal animation

Selected card flips to reveal game art
Shows game name, Metacritic, unlock cost
Game fills the empty shop slot

Step 4: Redraw option

"Don't like it? Redraw for 5 Keys"
Costs 5 keys to redraw (no escalation)
Completely new random roll from Pool 2
Can redraw unlimited times (5 keys each)

Draw Targeting (Weighted by Tier):
When drawing to fill a slot, weight the roll based on what tier was previously there:
If empty slot was CHEAP:

70% chance: Draw another Cheap game
20% chance: Draw Moderate game
10% chance: Draw Epic game (lucky!)

If empty slot was MODERATE:

20% chance: Draw Cheap game
60% chance: Draw another Moderate game
20% chance: Draw Epic game

If empty slot was EPIC:

10% chance: Draw Cheap game (unlucky)
20% chance: Draw Moderate game
70% chance: Draw another Epic game

Purpose: Maintains shop balance while allowing occasional surprises.

Generation:

ONLY earned by playing Pool 3 games for 30+ minutes
Amount earned = Game's Metacritic score

Usage:

Core Loop
Session Flow:
1. Start with 1 unlocked game + 5 locked shop games
2. Click unlocked game â†’ Generate Collection Power
3. Game drains after ~5-10 minutes of clicking
4. Unlock 1-2 cheap shop games with earned power
5. Click those new games â†’ They drain too
6. Now mostly stuck:
   - Not enough power for remaining expensive shop games
   - No keys to draw new games or refresh drained ones
7. PRESSURE POINT: Must either:
   A. Play a Key Game (30+ min on Steam) to earn keys
   B. Slowly grind minimal passive income (boring)
8. Player leaves site â†’ Plays "Hades" on Steam
9. Returns after 35 minutes:
   - +93 keys awarded
   - Hades moved Pool 3 â†’ Pool 2
   - All drained games auto-refreshed (free)
10. Spend keys:
    - Draw 2 new shop games (20 keys)
    - Redraw 1 (5 keys)
    - Unlock those with power
11. Continue clicking â†’ Repeat cycle
Target timing: 15-30 minutes of clicking before forced to leave and play.

Progression Balance
Early Game (0-30 minutes):

Unlock 2-3 cheap games
Drain all games
Hit key scarcity wall
Forced to play Key Game

Mid Game (1-5 hours):

Rotating between clicking and playing Key Games
Shop fills with moderate-tier games
Building up Game Library to 10-15 games
Keys become main bottleneck

Late Game (5+ hours):

Large Game Library (20+ games)
Epic games appearing in shop
Strategic choices: Draw new vs refresh drained vs unlock expensive
Key Games list shrinking as played

End Game (20+ hours):

Most games unlocked
Only highest-tier games left in shop
Key Games nearly exhausted
Victory condition: Unlock all games from Pool 2
